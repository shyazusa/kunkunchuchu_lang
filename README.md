# kunkunchuchu_lang

くんくんちゅっちゅ言語です

# Spec

Brainfuck系言語です

- *くんくん* - ポインタを右へ移動
- *ちゅっちゅ* - ポインタの指す値を1増やす
- *ぺろぺろ* - ポインタを左へ移動
- *ぴゅっぴゅ* - ポインタの指す値を1減らす
- *んっんっ* ポインタの指す値が0なら 対応する 'あっあっ' までジャンプする
- *あっあっ* ポインタの指す値が0で無いなら 対応する 'んっんっ' までジャンプする
- *ふっふっ* ポインタの指す値を出力する
- *むはむは* 入力から1バイト読み込こむ


それ以外の文字や改行は全て無視されます
